Abstractive Text Summarization Interface (BART, T5, PEGASUS, GPT)

This project provides an interactive Gradio-based interface for performing abstractive text summarization using multiple state-of-the-art transformer models:

	1.BART (facebook/bart-large-cnn)

	2.T5 (t5-base)

	3.PEGASUS (google/pegasus-xsum)

	4.GPT-2 (gpt2)

Additionally, the project visualizes METEOR, ROUGE-P, and ROUGE-1 evaluation scores using pie charts.
